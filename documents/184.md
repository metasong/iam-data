## Variable
### Declearation, initialization and assignment
```S
a:i32 // declearation
a; // declear a variable with its type infered by following usage. ';' is needed.
// declear and assignment
a:7i32 // 7 is initial value: a:i32=7, type is infered
// := operator is used for content assignment
a = 6i32  // assignment to an already declared variable

// allow variable overriding
a:3 // declear a,  3 is int32 if not 3i64
a:6 // error: in same scope, try to override the previous declearation with a new declearation of the same name
{
    a:7 // new overriding declearation in the block scope, new a
}

```

```
   var usually is a discriminated union defined with '|'

   && and ||: lazy boolean expression operator
   & and |: 'and' and 'or' pattern mach oparator
```
